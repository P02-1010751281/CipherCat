# Algorithm 3  π(A)

**章节**: §3.2.3  
**类别**: KECCAK-p 步映射 — Lane 位置置换

### 规范

```
Input:  state array A
Output: state array A′

1: For all (x,y,z):
       A′[x,y,z] = A[(x+3y) mod 5, x, z]
2: Return A′
```

### 备注

重新排列 lane 位置: (x+3y mod 5, x) ← (x,y)。
等效于 B[y, 2x+3y mod 5] = ROT(A[x,y], r[x,y])。
