# Algorithm 5  rc(t)

**章节**: §3.2.5  
**类别**: 轮常数生成（线性反馈移位寄存器）

### 规范

```
Input:  integer t
Output: bit rc(t)

1: If t mod 255 = 0, return 1
2: Let R = 10000000 (8-bit)
3: For i from 1 to t mod 255:
   a. R = 0 || R
   b. R[0] = R[0] ⊕ R[8]
   c. R[4] = R[4] ⊕ R[8]
   d. R[5] = R[5] ⊕ R[8]
   e. R[6] = R[6] ⊕ R[8]
   f. R = Trunc₈[R]
4: Return R[0]
```

### 备注

LFSR 生成轮常数的单个 bit。
CipherCat 中直接硬编码 24 个 64-bit 轮常数，避免运行时计算。

### CipherCat

内嵌于 `keccak_f1600`（预计算为 RC[24] 常量数组）
