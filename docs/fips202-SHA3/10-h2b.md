# Algorithm 10  h2b(H, n)

**章节**: Appendix B  
**类别**: 十六进制 → 比特串

### 规范

```
Input:  hex string H of 2m digits;  n ≤ 8m
Output: bit string S of length n

1: Parse H = H₀H₁…H_{2m−1}
2: For i = 0 to m−1:
   a. hᵢ = 16·H₂ᵢ + H₂ᵢ₊₁
   b. bᵢ₀…bᵢ₇ = binary representation of hᵢ (little-endian)
3: T[8i+j] = bᵢⱼ
4: Return S = Trunc_n(T)
```

### 备注

将十六进制字符串转换为比特串。
示例: H=0xA32E, n=14 → T=1100010101110100, S=11000101011101。
