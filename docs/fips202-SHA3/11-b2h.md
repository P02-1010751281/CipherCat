# Algorithm 11  b2h(S)

**章节**: Appendix B  
**类别**: 比特串 → 十六进制

### 规范

```
Input:  bit string S of length n
Output: hex string H of 2·ceil(n/8) digits

1: n = len(S)
2: T = S ‖ 0^{−n mod 8}, m = ceil(n/8)
3: For each byte: bᵢⱼ = T[8i+j]
4: For i = 0 to m−1:
   a. hᵢ = Σ bᵢⱼ·2^j
   b. H₂ᵢ H₂ᵢ₊₁ = hex digits of hᵢ
5: Return H = H₀…H_{2m−1}
```

### 备注

h2b 的逆运算。不足整字节时补零 (0-pad to byte boundary)。
